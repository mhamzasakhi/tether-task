# signature pharmacy
## Setup
install: <br/>
````
npm install
````
configure: <br/>
````
fill all environment variables to .env file
````
Create database: <br/>
````
node_modules\.bin\sequelize db:create
````
running migrations: <br/>
````
node_modules\.bin\sequelize db:migrate
````
## Starting app
default mode
````
npm run start
````
development mode with watcher
````
npm run dev
````
